    Create `backend/src/core/celery_app.py`.
    Initialize a Celery app named 'aria' with Redis as the broker and result backend.
    
    ```python
    from celery import Celery
    import os

    REDIS_URL = os.getenv("REDIS_URL", "redis://localhost:6373/0")

    celery_app = Celery(
        "aria",
        broker=REDIS_URL,
        backend=REDIS_URL,
        include=["src.services.tasks"]
    )

    celery_app.conf.update(
        task_track_started=True,
        task_serializer="json",
        result_serializer="json",
        accept_content=["json"],
    )
    ```
    Note: Ensure REDIS_URL matches your local/docker setup. Defaulting to 6379 or specified env.
  </action>

    - When a WebSocket connects for `workspace_id`, ensure a subscriber is listening for `workspace:{workspace_id}`.
    - Use `redis.asyncio` for non-blocking subscription.
    
    ```python
    import aioredis
    
    async def redis_listener(self):
        pubsub = self.redis.pubsub()
        await pubsub.psubscribe("workspace:*")
        async for message in pubsub.listen():
            if message['type'] == 'pmessage':
                channel = message['channel'].decode()
                workspace_id = channel.split(":")[1]
                data = json.loads(message['data'].decode())
                await self.send_status(workspace_id, data)
    ```
